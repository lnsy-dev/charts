Testing dataroom-chart with src attribute

Instructions:

  1. Make sure you have built the project (e.g., with npm run build). This will create the dist/main.js file.
  2. Run a simple web server from your project root. For example, if you have Python 3, run: python3 -m http.server
  3. Open your browser and go to http://localhost:8000/test-src.html to see the chart.
